# Youth Market Lane

Youth Market Lane 是一个面向青年市集、社团展位与活动筹备协同的平台，面向真实业务协作场景提供完整的仪表盘、列表页、详情页、录入页面和接口层。仓库采用前后端一体的中型工程结构，既能直接作为 Docker 化示例运行，也方便继续扩展权限、报表、通知、审批和集成能力。

## 适用场景

- 团队需要围绕固定主题管理事项、状态、时间线和协同备注。
- 管理者需要通过首页快速查看当前规模、关键指标和重点问题。
- 业务人员需要通过 Web 页面和 API 同时维护与消费数据。

## 核心能力

- 首页仪表盘展示整体业务摘要和重点统计。
- 列表页支持浏览核心对象，并提供筛选或检索能力。
- 详情页支持查看结构化字段、阶段状态和关联事项。
- 新建页面支持录入数据，部分项目包含实时预览或辅助面板。
- 内置种子数据，容器首次启动后即可进入可用状态。
- 提供可直接联调的 JSON API，便于继续集成外部系统。

## 技术栈

- 后端：Java / Spring Boot；模板：Thymeleaf；前端：原生 CSS；数据库：H2

## 目录结构

- app/、src/ 或 cmd/：后端入口、路由、服务和数据访问逻辑。
- templates/、views/ 或 web/：页面模板与前端展示层。
- static/ 或 public/：样式、脚本等静态资源。
- data/：本地数据库或运行期数据目录。
- tests/：预留测试或示例测试目录。
- patch/：预留给后续补丁产物的目录。

## Docker 启动

    docker build -t public-youth-market-lane .
    docker run --rm -p 8080:8080 -p 2222:22 public-youth-market-lane

容器内工作目录固定为 /app，并通过 SSH 插件开放 22 端口，方便在远程容器中继续开发或调试。

## 本地运行

请参考仓库中的 start.sh 作为启动入口；如果需要本地开发，可按项目使用的语言安装依赖后直接执行对应启动命令。
